## Installation

-Masuk Ke Folder Client

-Buka Terminal

-jalankan

```bash
  npm install 

  npm run serve
```
    
Masuk Ke Folder Server

-Buka Terminal

-jalankan

```bash
  npm install 

  npm run dev
```
